1. 모티베이션 & 인트로

1.1 중간발표 알고리즘을 그대로 쓰거나 발전시키지 않은 이유

중간발표에서 제안한 알고리즘은 UWB와 WiFi 두 이종(heterogeneous) 센서의 측정 합의도(agreement)를 기반으로 가중치를 조정하는 SAFL 계열 구조였다. 그 설계의 핵심 전제는 "서로 다른 물리 특성을 가진 두 센서가 동시에 같은 위치를 지지할 때 신뢰도가 높다"는 이종 센서 융합 가정이었다.

그러나 Final Project 데이터는 18개 기지국이 모두 동일한 RTT 거리만 제공하는 단일(homogeneous) 측정 구조다. 이종 센서가 존재하지 않으므로 SAFL의 합의도 가정 자체가 성립하지 않는다. 중간발표 아이디어를 "발전"시키는 것이 아니라 적용 자체가 불가능하다는 것이 첫 번째 이유다. 둘째로, 두 과제의 측정 환경이 정량적으로 완전히 다르다. 아래 표는 두 환경의 핵심 차이를 정리한 것이다.

| 항목 | 중간발표 환경 | Final Project 환경 |
|---|---|---|
| 센서 구성 | UWB + WiFi (이종) | RTT 단일 (동종) |
| 공간 규모 | 약 12 m × 17 m | 약 100 m × 40 m |
| 측정 편향(bias) 규모 | 수 m 수준 | 평균 +15.9 m |
| 적용 가능 핵심 아이디어 | 이종 센서 합의 | 적용 불가 |

따라서 중간발표 산출물을 재사용하는 대신, Final 데이터의 특성을 처음부터 다시 분석하여 단일 RTT 환경에 맞는 알고리즘을 새로 설계했다.

1.2 중간발표에서 받은 피드백의 수용

중간발표에서 받은 지적은 새 설계와 본 보고서에 다음과 같이 반영했다.

| 피드백 | 반영 내용 |
|---|---|
| ① 제목이 부적절 | 알고리즘의 두 핵심(결정적 consensus + NLOS 상한 제약)을 그대로 드러내는 제목으로 교체 |
| ② 순서도가 미흡 | 본 보고서는 이미지 첨부가 금지되므로, §2에서 각 단계의 입력·연산·출력을 수식과 함께 빠짐없이 서술하여 그림 없이도 구현이 가능하도록 했다 |
| ②-2 합산 기호에 인덱스가 없어 WiFi/UWB 구분이 불명확 | 단일 동종 RTT 구조로 재설계하여 센서 종류 구분 자체가 사라졌고, 모든 합산은 18개 기지국에 대한 명시적 인덱스 i로 표기했다(§2) |
| ③ 불필요한 과정 제거 | 모든 구성요소를 이론·실험으로 감사하여 효과가 입증되지 않은 요소(Huber 손실, 최대유효거리 필터, Lower-K 사전선별)를 제거했다(§4) |
| ④ Huber 손실·격자탐색·직렬화 적용 검토 | 직렬(serial) 구조를 채택했고, Huber 손실은 실제로 적용해 검증한 뒤 제거했으며(작동하지 않음, §4), 격자탐색 대신 해석적 closed-form 삼변측량을 사용해 이산화 오차와 추가 파라미터 없이 정확해를 구한다 |
| ⑤ 결과의 의미 해석 요구 | 단순 수치 비교가 아니라, 각 수치가 무엇을 의미하고 어떤 설계 판단을 뒷받침하는지 §4에서 적극적으로 해석했다 |

1.3 데이터 특성에서 알고리즘이 도출된 흐름

정답 위치를 이용해 제공된 700명 데이터를 분석한 결과는 다음과 같다.

| 특성 | 값 |
|---|---|
| 공간 크기 | 약 100 m × 40 m (기지국 3행 6열 배치) |
| 측정 거리 평균 / 최대 | 63.8 m / 307.3 m |
| NLOS bias 평균 / 중앙값 | +15.93 m / +10.15 m |
| bias > 0 비율 | 81.4 % |
| bias > 5 m 비율 | 59.0 % |
| \|bias\| < 1 m 비율 | 29.9 % |
| 모든 UE의 LOS급 측정 보유 | 700 / 700 (각 UE가 최소 1개) |
| 결측(NaN) | 0 개 |

이 분포에서 알고리즘 설계 전체를 지배하는 단 하나의 물리적 사실이 도출된다. 신호가 장애물에 막히면 우회·반사 경로를 거치므로 측정 거리는 실제 거리보다 길어진다(양의 편향)(Nkrow et al., 2024). 즉 측정값 d̂는 참 거리 d를 거의 항상 과대평가한다(d̂ ≥ d). 다만 bias > 0 비율이 100 %가 아니라 81.4 %인 것은, NLOS 양의 편향 위에 타임스탬프 분해능·클록 동기화 등에서 비롯되는 대략 영평균(zero-mean)의 측정 잡음이 더해지기 때문이다. NLOS 성분이 거의 없는 LOS급 측정에서는 이 잡음이 가끔 음의 값을 만들어 d̂ < d 가 되며, 음의 편향 18.6 %는 대부분 이런 LOS 근처 측정이다. 이 점은 뒤의 §2.4에서 상한 제약을 inlier에 부과하지 않는 이유의 직접적 근거가 된다(LOS 측정에 섞인 영평균 잡음의 음의 부분까지 잘라내면 추정이 안쪽으로 편향되므로). 위의 핵심 사실(d̂ ≥ d)은 두 개의 독립적인 설계 원리를 낳는다.

- 원리 A (작은 값 = LOS 후보): 편향이 양수이므로, 상대적으로 작은 측정값일수록 직접경로(LOS)일 가능성이 높다. 이는 신뢰할 만한 측정 부분집합을 식별하는 consensus 단계의 근거다.
- 원리 B (측정값 = 상한): d̂ ≥ d 는 곧 ‖x − BS_i‖ ≤ d̂_i 를 의미한다. 즉 UE는 각 기지국에서 반지름 d̂_i 인 원판(disc) 안에 있어야 한다. 이는 위치를 물리적 가능 영역으로 제약하는 근거다.

같은 물리 사실을 한 번은 "부분집합 선택"에, 또 한 번은 "위치 제약"에 사용한다는 점이 본 알고리즘의 중심 아이디어다.

1.4 알고리즘 high-level 소개

알고리즘은 무작위성이 없는 결정적(deterministic) 3단계 직렬 구조이며 기계학습을 사용하지 않는다. (1) 물리 필터로 탐색 범위를 정하고 명백히 무효한 측정을 제거한 뒤, (2) 모든 3개 측정 조합의 삼변측량을 전수 평가하여 가장 많은 측정과 일치하는(inlier 최대) 위치를 consensus로 선택하고, (3) 선택된 inlier로 최소제곱 정밀화를 수행하되 consensus가 버린 측정들을 원리 B의 상한 제약으로 재활용한다. 핵심 성과는 평균오차(MAE) 1.39 m, 2 m 이내 도달률 91.0 %이다.

2. 알고리즘 설명

본 절은 수식과 자연어만으로 알고리즘을 기술하며, 이 설명만으로 코드 구현이 가능하도록 모든 연산을 명시한다.

2.1 표기

기지국 좌표를 BS_i = (x_i, y_i), i = 1, …, M (M = 18)로 두고, 한 UE에 대한 측정 거리 벡터를 d̂ = (d̂_1, …, d̂_M)로 둔다. 추정 위치를 x = (x, y) ∈ ℝ² 로 표기한다. 점 x와 기지국 i 사이 거리는 ρ_i(x) = ‖x − BS_i‖, 잔차는 r_i(x) = ρ_i(x) − d̂_i 로 정의한다. 고정 상수는 inlier 임계값 τ = 1.5 m와 탐색 여유 비율 β = 0.30 두 개뿐이다.

2.2 Stage 1 — 물리 필터

기지국 좌표의 좌표축별 최소·최대를 x_min, x_max, y_min, y_max 라 하면, 탐색 경계는 기지국 배치 범위에 양쪽으로 β의 여유를 둔 직사각형으로 정의한다.

  B = [ x_min − β·W_x ,  x_max + β·W_x ] × [ y_min − β·W_y ,  y_max + β·W_y ]

여기서 W_x = x_max − x_min, W_y = y_max − y_min 이다. 경계가 기지국 좌표에서 자동으로 산출되므로 공간 의존 상수를 손으로 정하지 않는다. 유효 측정 집합은 유한하고 양수인 측정만으로 정의한다.

  V = { i : d̂_i is finite and d̂_i > 0 }

만약 |V| < 3 이면 2차원 위치를 결정할 수 없으므로 기지국 중심 (1/M) Σ_{i=1}^{M} BS_i 를 반환하고 종료한다.

2.3 Stage 2 — 결정적 consensus

유효 측정을 거리 오름차순으로 정렬한 인덱스 순서를 π라 한다(d̂_{π(1)} ≤ d̂_{π(2)} ≤ ⋯). 유효 측정 중 서로 다른 3개의 모든 조합 a, b, c ⊂ V 에 대해 다음을 수행한다.

(i) Closed-form 삼변측량. 세 기지국 (x_a, y_a), (x_b, y_b), (x_c, y_c)와 측정 d̂_a, d̂_b, d̂_c 가 주어졌을 때, 원 방정식 ρ_j(x)² = d̂_j² 세 개를 쌍으로 빼면 비선형 항이 소거되어 선형 연립방정식이 된다(이는 원 방정식을 쌍으로 빼 선형화하는 표준 삼변측량 기법이며, 본 연구의 기여는 이 해법 자체가 아니라 §2.3 전체에 걸친 consensus 결합에 있다). 첫 식에서 둘째·셋째 식을 빼면

  2(x_b − x_a)·x + 2(y_b − y_a)·y = (d̂_a² − d̂_b²) − (x_a² + y_a²) + (x_b² + y_b²)
  2(x_c − x_a)·x + 2(y_c − y_a)·y = (d̂_a² − d̂_c²) − (x_a² + y_a²) + (x_c² + y_c²)

이를 두 미지수 (x, y)에 대한 선형 연립방정식 A·x = g 로 둘 수 있다. 계수 행렬 A의 두 행은 각각 ( 2(x_b − x_a),  2(y_b − y_a) )와 ( 2(x_c − x_a),  2(y_c − y_a) )이고, 우변 벡터 g의 두 성분은 위 두 식의 우변, 즉 d̂_a² − d̂_b² − x_a² − y_a² + x_b² + y_b² 와 d̂_a² − d̂_c² − x_a² − y_a² + x_c² + y_c² 이다.

후보 위치는 x_cand = A⁻¹ g 로 구한다. A가 특이(세 기지국이 일직선)이면 이 조합은 건너뛴다.

(ii) 경계 게이팅. x_cand ∉ B 이면 비현실적 후보로 보고 건너뛴다.

(iii) Inlier 계수. 후보에 대해 전체 유효 측정의 잔차를 계산하고 임계값보다 작은 측정의 개수를 센다.

  n(x_cand) = | { i ∈ V : |ρ_i(x_cand) − d̂_i| < τ } |

(iv) 선택. n(·)이 최대인 후보를 선택한다. 동점(inlier 수가 같은 후보가 둘 이상)인 경우, 측정 거리 오름차순으로 정렬한 anchor 순서 π를 따라 조합을 사전식(lexicographic)으로 열거했을 때 가장 먼저 나온 조합을 유지한다(동점에서는 후보를 갱신하지 않으므로, 즉 strict-greater 비교로 최초의 최대 후보가 남는다). 사전식 순서에서 앞선 조합일수록 거리가 작은(LOS 후보) anchor를 우선하므로, 이로써 원리 A(작은 값 = LOS)가 동점 처리에 반영된다. 이 규칙은 입력에 대해 출력을 유일하게 정하므로, 전수 열거와 결정적 동점 처리에 의해 무작위성이 전혀 없다. 어떤 후보도 경계 안에 없으면 가장 작은 측정 기지국 위치를 임시 후보로 둔다.

선택된 위치 x* 에서 inlier 집합과 그 여집합(consensus가 버린 측정)을 정의한다.

  I = { i ∈ V : |r_i(x*)| < τ } ,  O = V ∖ I

만약 |I| < 3 이면 정밀화에 필요한 최소 3개를 확보하기 위해 잔차가 가장 작은 3개 측정을 I로 둔다.

2.4 Stage 3 — NLOS 상한 제약 정밀화

inlier 집합으로 최소제곱 목적함수를 세우되, consensus가 버린 측정 O를 원리 B의 단방향 상한으로 재활용하여 제약을 부과한다.

  min_{x ∈ B}  Σ_{i ∈ I} ( ρ_i(x) − d̂_i )²    s.t.  ρ_i(x) ≤ d̂_i   ∀ i ∈ O

이 제약은 "참 거리는 측정값을 넘지 않는다(d̂ ≥ d)"는 물리 사실의 직접 표현이며, 별도의 가중치 파라미터를 요구하지 않는다(파라미터-free). 상한 부등식은 원칙적으로 모든 유효 측정에 성립하지만(d̂_i ≥ d_i, 즉 ∀ i ∈ V), 제약은 consensus가 버린 측정 O에만 부과한다. 이유는 두 가지다. 첫째, inlier I는 목적함수가 이미 ρ_i ≈ d̂_i 가 되도록 강하게 끌어당기므로 상한을 자연히 충족하여 제약을 명시하는 것이 중복이다. 둘째, LOS급 측정은 d̂_i ≈ d_i 여서 상한이 거의 등호에 가깝고, 여기에 부등식을 강제하면 LOS 측정에 섞인 작은 음의 잡음까지 잘라내 추정을 안쪽으로 편향시킨다. 실제로 제약을 inlier를 포함한 전체 V에 적용하면 MAE가 오히려 악화됨을 실험으로 확인했다. 제약은 추정 위치가 어떤 기지국의 측정 반경을 벗어나려 할 때만 작동하므로, 이미 잘 측위된 UE는 전혀 건드리지 않고 정보가 부족하여 발산하려는 UE만 물리적 가능 영역(원판들의 교집합) 안으로 끌어당긴다.

이 문제는 순차 이차계획법(SLSQP)으로 푼다. 초기값은 x* 이다. 드물게 SLSQP가 수렴에 실패하는 경우(실측상 약 1 %), 동일 제약을 큰 가중치 w의 외부 벌점(exterior penalty)으로 완화한 다음 식을 L-BFGS-B로 풀어 대체한다.

  min_{x ∈ B}  Σ_{i ∈ I} ( ρ_i(x) − d̂_i )² + w · Σ_{i ∈ O} [ max(0, ρ_i(x) − d̂_i) ]²

결과는 w가 일정 수준(약 16) 이상이면 사실상 변하지 않으므로 w는 조정 대상 상수가 아니라 제약을 강제하기 위한 충분히 큰 값으로 둔다(w = 50). 이 폴백 덕분에 어떤 입력에서도 결과가 단순 경계 최소제곱보다 나빠지지 않는다.

2.5 선행 연구와의 관계 및 차이

본 알고리즘은 세 갈래의 선행 연구와 닿아 있으며, 각 부분에서 무엇이 기존 개념이고 무엇이 본 연구의 제안인지는 다음과 같다.

| 구성 | 기존 연구가 제안한 것 | 본 연구가 제안/변경한 것 |
|---|---|---|
| Consensus | RANSAC(Fischler & Bolles, 1981): 무작위 최소표본 추출 후 inlier 합의로 모델 추정 | 18개 측정의 모든 3-조합을 전수 평가하여 무작위성을 제거한 결정적 consensus(M = 18에서 조합 수가 작아 실현 가능) |
| 표본 우선순위 | PROSAC(Chum & Matas, 2005): 품질 척도로 정렬해 우선순위 높은 표본부터 점진 추출 | 거리 오름차순 정렬을 동점 처리 규칙으로만 사용(점진 추출 아님). 거리 절단(Lower-K) 사전선별은 검증 후 제거(§4) |
| NLOS 상한 | NLOS 과대평가로 인해 참 위치가 측정 반경 원판들의 교집합에 있다는 feasibility 개념(예: Yousefi 등, 2014, 이동체 트래킹용 제약 Square-Root UKF로 정식화) | 동일 물리 원리를, consensus가 폐기한 측정에 한해 단방향 상한 제약으로 재활용. 원전이 시계열 이동체 트래킹용 동적 칼만필터인 데 비해 본 연구는 단일 시점(static) 위치 추정에 맞추며, 무거운 QCQP/SDP나 칼만필터가 아니라 SLSQP(+벌점 폴백)로 결정적·경량 구현 |

요컨대 표준 RANSAC은 outlier를 버리고, 집합 멤버십(set-membership) 계열(예: Yousefi 등, 2014)은 모든 측정을 제약으로 쓴다. 본 연구는 그 사이의 새로운 결합을 제안한다. 즉 inlier는 최소제곱으로 적합하고, 버려질 outlier는 폐기 대신 단방향 상한 제약으로 강등하여 정보를 끝까지 활용한다.

3. Agent AI 활용 방안

본 프로젝트에서 본인은 항법·무선측위 분야의 전문 지식을 사전에 충분히 보유하지 않았다. 그래서 여러 Agent AI를 도구로 사용하되, 무엇을 받아들이고 무엇을 버릴지에 대한 판단과 최종 결정은 전적으로 본인이 수행하는 방식으로 진행했다. AI는 지식 공급원이자 실험 실행기였고, 본인은 문제 정의자이자 검증·결정의 주체였다.

3.1 사용한 Agent AI와 역할

- Claude (대화형 + 코드 실행): 위치추정에 쓰이는 여러 추정기(최소제곱, M-estimator, RANSAC 계열 등)와 최적화 기법을 소개받았고, 본인이 지정한 실험(절제 연구, 교차검증, 구성요소 감사, 음성 결과 검증)을 코드로 실행·재현하는 데 사용했다.
- GPT (심층 분석): 완성된 알고리즘의 가정(특히 "NLOS는 거리를 키운다")이 문헌·물리 사실과 부합하는지 심층 분석과 사실성 검토를 받았다.
- Gemini (개선 제안): 성능 향상 후보로 기하 배치(GDOP) 기반 표본 가중, 동적 임계값(adaptive τ), 그리고 NLOS 상한 제약(완화 최소제곱) 세 가지를 제안받았다.

3.2 본인과 AI의 역할 구분

| 작업 | Agent AI의 역할 | 본인의 역할 |
|---|---|---|
| 도메인 지식 습득 | 추정기·최적화 기법의 원리 설명 제공 | 어떤 기법이 이 데이터에 적절/부적절한지 기준 수립 |
| 데이터 특성 분석 | 분석 코드 실행 | 분석 항목 지정, "양의 편향" 사실에서 두 설계 원리 도출 |
| 구성요소 취사선택 | 절제 연구 수치 산출 | 제거·유지 기준(이론+실험 동시 충족) 정의 및 판정 |
| 외부 분석 검토 | GPT가 한계점 제시 | 제시된 한계의 타당성을 직접 재검증(예: 음의 편향 18.6 % 우려를 실측으로 비이슈 확인) |
| 개선 아이디어 평가 | Gemini가 3개 제안 | 3개를 동일 교차검증으로 전부 정량 시험 → 2개 기각, 1개(상한 제약) 채택 |
| 최종 설계 결정 | 비교 자료 제공 | 채점 요건(성능·참신성·실행시간)을 종합한 최종 구조 확정 |

특히 강조할 점은, AI의 제안을 그대로 수용하지 않았다는 것이다. Gemini가 제안한 세 기법은 모두 그럴듯했으나 본인이 동일한 교차검증 절차로 직접 시험했고, 그 결과 GDOP 가중과 동적 임계값은 효과가 없거나 오히려 해로워 기각했으며, 상한 제약만이 채택 기준을 통과해 받아들였다. 즉 AI는 가설을 제공했고, 가설의 진위 판별과 채택 결정은 본인의 검증 절차가 수행했다.

4. 결과 도출 & 디스커션

4.1 최종 성능

제공된 700명 전체에 대한 최종 성능과 5-fold 교차검증 결과는 다음과 같다.

| 지표 | 700명 전체 | 5-fold 평균(표준편차) |
|---|---|---|
| MAE | 1.393 m | 1.393 m (0.285) |
| 중앙값(Median) | 0.619 m | — |
| RMSE | 3.476 m | — |
| 1 m 이내 | 75.6 % | — |
| 2 m 이내 | 91.0 % | 91.0 % (2.9) |
| 5 m 이내 | 95.1 % | — |
| 샘플당 처리시간 | 약 14.5 ms | — |

폴드 간 표준편차가 작고 평가셋과 평균이 일치하므로, 같은 1000명 모집단에서 떼어낸 hidden 300명에서도 유사한 성능이 기대된다. 다만 hidden 분포가 미세하게 다를 가능성까지 완전히 배제할 수는 없으므로 이는 단정이 아니라 시사 수준으로 본다. hidden 300명 기준 예상 총 실행시간은 약 4.4초로, 10분 제한에 비해 충분한 여유가 있다.

4.2 각 구성요소의 역할과 필수성 — 이론·실험 검증

설계 원칙은 "이론과 실험 양면에서 필요성이 입증된 구성요소만 유지한다"였다. 각 구성요소를 하나씩 제거했을 때의 MAE 증가량(ΔMAE, 양수일수록 필수)을 통해 필요성을 정량 검증했다.

| 구성요소 | 제거 시 ΔMAE | 이론적 역할 | 판정 |
|---|---|---|---|
| Inlier 필터 + consensus | +19.1 m | 거친 오차(gross error) 비율이 약 60 %로, 어떤 동변(equivariant) 추정기도 보장할 수 있는 breakdown 한계(50 %)를 넘어선다. 이는 강건손실·가중만으로는 한계가 있음을 시사하며, 실제로 inlier 필터를 제거하고 전체 측정에 직접 추정하면 MAE가 20 m를 초과한다(본 표의 +19.1 m) | 필수(최우선) |
| NLOS 상한 제약 | +2.334 m | 버려진 측정의 물리적 상한 정보로 정보부족 UE의 발산을 억제 | 필수 |
| 정밀화 단계 | 1 m 이내율 약 17 %p 하락 | 3점 삼변측량은 3개만 쓰므로, 전체 inlier로 재추정해야 정밀해짐 | 필수 |

또한 다음 보조 장치는 드물게 작동하지만 미정의·발산을 막는 이론적 역할이 있어 유지한다. 유효측정 3개 미만 시 중심 반환(정보 부족 시 정의된 동작), inlier 3개 미만 시 잔차 최소 3개 사용(2차원 위치에 최소 3개 필요), 정밀화 경계 제약(드물게 작동하는 발산 방지).

4.3 의도적으로 제거한 구성요소

설계 초기에는 포함했으나 검증 결과 필요성이 입증되지 않아 제거한 요소들이다. 이는 "불필요한 과정 제거"라는 피드백과 군더더기 없는 설계 원칙에 따른 것이다.

| 제거한 요소 | 실험 근거 | 이론 근거 |
|---|---|---|
| Huber 강건손실 (Huber, 1964) | inlier 잔차 최댓값이 1.93 m로 Huber 전환점(5.0 m)을 700명 전원·5-fold 전 폴드에서 한 번도 넘지 않아, 최소제곱과 소수점 셋째자리까지 동일 | inlier 필터가 이미 outlier를 제거하므로 남은 집합에는 outlier가 없고, 그 경우 강건손실이 줄 이점이 없다(잔차가 영평균에 가까운 잡음이면 일반 최소제곱으로 충분하다). 강건손실은 "outlier가 남은 집합"용 도구라 여기서는 작동할 여지 자체가 없으며, 위 실험이 이를 직접 보여준다 |
| 최대유효거리 필터 | 제거 시 ΔMAE −0.006 m(노이즈 수준, 오히려 미세 개선) | 큰(NLOS) 측정 차단 역할은 거리 정렬 동점 처리와 inlier 필터가 이미 수행하므로 고유 기여가 없다(중복) |
| Lower-K 사전선별 | 상한 제약 도입 후 제거 시 ΔMAE −0.004 m이며 1 m 이내율은 오히려 +2.0 %p 상승 | 상한 제약이 발산을 잡아주면서 consensus 후보 선택의 질에 대한 민감도가 사라져, 작은 K개로 좁히는 사전선별의 정확도 기여가 소멸. 남은 효과는 연산량 절감뿐인데 본 과제의 실행시간 여유를 고려할 때 불필요하며, 파라미터 K를 제거하는 편이 더 단순하다. 단 "작은 값이 LOS"라는 물리적 통찰 자체는 버리지 않았고, K개로 절단하는 하드 필터 대신 전수 조사 후 동점 시의 결정적 타이브레이커 규칙(§2.3)으로 강등·재활용하여 파라미터 K만 소거했다 |

초기 PROSAC + Huber 구조(MAE 3.176 m)에서 출발하여, 위의 제거와 상한 제약 도입을 거쳐 최종 MAE 1.393 m에 도달했다. 즉 성능 개선은 "기능을 더한 것"이 아니라 "불필요한 것을 덜어내고 물리적으로 옳은 제약 하나를 더한 것"의 결과다.

4.4 비교의 공정성(Fairness)

성능을 다른 접근과 비교할 때 비교가 공정한지 따지는 것이 수치 자체보다 중요하다.

(a) 기계학습 대비 비교. 흔히 "딥러닝 대 단순 삼변측량" 식의 비교는 학습 데이터·가정이 달라 불공정해지기 쉽다. 이를 피하기 위해, 18개 측정에서 위치를 직접 회귀하는 모델(RandomForest 등)을 본 알고리즘과 동일한 5-fold 교차검증으로 비교했다. 학습 모델은 학습 폴드에서만 적합하고 held-out 폴드에서만 평가하여 정보 누출을 차단했다.

| 방법(동일 5-fold CV) | MAE | 2 m 이내 |
|---|---|---|
| 본 알고리즘 | 1.393 m | 91.0 % |
| RandomForest 회귀 | 7.606 m | 6.3 % |
| NLOS 편향 보정(거리 의존) 후 측위 | 16.118 m | 2.0 % |

학습 모델이 진 것은 우리가 불리하게 설정해서가 아니라, 종단간 회귀가 기하 구조와 물리 제약을 활용하지 못하기 때문이다. 특히 회귀 모델은 그럴듯한 영역으로 평균 회귀할 뿐 2 m 이내 정밀 측위를 거의 하지 못한다(91.0 %→6.3 %). 또한 편향 보정은 양·음 편향이 섞인 분포에서 LOS 측정까지 오염시켜 consensus의 재료를 망가뜨리므로 더 악화된다. 두 결과 모두 "consensus가 이미 NLOS를 처리하고 상한 제약이 정보를 활용하므로, 보정·학습 기법이 추가로 기여할 자리가 없다"는 결론을 동일 절차의 공정 비교로 뒷받침한다.

(b) 문헌값과의 비교. 일부 WiFi RTT 측위 연구는 약 1.1 m 수준의 평균오차를 보고하나, 그러한 결과는 편향이 작은(sub-meter 수준) 환경에서 얻어진 것이다. 본 데이터는 평균 편향 +15.9 m, 5 m 초과 편향 59 %의 극단적 NLOS 환경이므로 문헌값과의 직접 비교는 공정하지 않다. 같은 척도로 본다면, LOS 정보가 충분한 UE에서의 중앙값 0.62 m는 환경 난이도 대비 경쟁력 있는 수준이다.

(c) 자체 평가 방식의 공정성. 모든 비교와 채택 결정은 5-fold 교차검증과 학습/평가 분리를 전제로 했고, 개선 채택 기준을 사전에 "MAE 0.2 m 이상 또는 2 m 이내 1 %p 이상 개선"으로 고정했다. 상한 제약만이 이 기준을 큰 폭으로 통과했고(고립 기여 ΔMAE +2.334 m, §4.2), 나머지 후보는 모두 기준 미달로 기각했다. 참고로 상한 제약을 도입한 최종 구조는 초기 PROSAC + Huber 구조(MAE 3.176 m, 2 m 이내율 86 %) 대비 1.393 m·91 %에 도달했는데, 이 전체 개선은 상한 제약 단독이 아니라 구조 단순화(PROSAC→결정적 전수 consensus, Huber→최소제곱 등)까지 포함한 결과이며, 상한 제약 단독의 고립 기여는 위의 ΔMAE +2.334 m이다.

4.5 본인의 사고·구현의 적절성

본 프로젝트에서 본인의 기여는 새로운 추정기를 발명한 것이 아니라, 데이터의 단일 물리 사실을 끝까지 일관되게 활용하도록 설계를 좁히고, 모든 구성요소를 검증으로 정당화한 것이다. 특히 "더 개선할 것이 없다"는 잠정 결론을 외부 제안(상한 제약)의 검증으로 스스로 뒤집은 과정은, 결론보다 검증을 우선하는 태도가 실제로 성능 향상으로 이어졌음을 보여준다.

본 연구의 신규 기여를 한 곳에 모으면 다음과 같다. (1) RANSAC의 inlier 최대화를 M = 18에 맞춰 모든 3-조합 전수 평가로 바꾼 무작위성 없는 결정적 consensus, (2) RANSAC이 폐기하고 set-membership 계열이 모두 제약으로 쓰는 두 극단 사이에서, 폐기될 outlier만 골라 단방향 상한 제약으로 강등·재활용하는 결합, (3) 무거운 QCQP/SDP 대신 SLSQP(+벌점 폴백)로 구현한 파라미터-free·경량 정밀화. 기존 개념과 본 연구 기여의 항목별 대조는 §2.5, 각 문헌의 구체적 활용은 §5에 정리하였다.

4.6 장점·단점·향후 연구

장점. (1) 무작위성이 없어 같은 입력에 항상 같은 출력을 주므로 채점 재현성이 보장된다. (2) 공간 의존 상수가 0개이고 조정 상수가 τ, β 둘뿐이라 다른 InF 배치로 이식하기 쉽다. (3) 같은 물리 사실을 부분집합 선택과 위치 제약에 이중으로 활용하여, 버려질 측정까지 정보로 쓴다. (4) 전수 평가라고 해도 본 InF 표준 배치(M = 18)에서 평가 대상 조합은 C(18, 3) = 816개에 불과하여, 무작위성을 완전히 제거하면서도 샘플당 약 14.5 ms의 실시간성을 달성한다.

단점. (1) 상한 제약의 신뢰성은 d̂ ≥ d 가 거의 성립하는 환경에 의존한다. 측정 잡음이 커서 d̂ < d 가 빈번한 환경에서는 하드 제약보다 벌점형 완화가 더 안전하다. (2) 전수 조합 평가는 기지국 수가 많아지면 C(M, 3)으로 증가하므로, 대규모 망에서는 정렬 기반 부분 후보로 제한하는 확장이 필요하다. (3) 2차원 평면 가정에 한정된다.

향후 연구. 기지국 수 증가에 대비한 후보 조합의 적응적 축소, 측정 잡음이 큰 실측 환경에서의 하드/벌점 제약 자동 전환, 그리고 동적(이동) UE에 대한 시간적 평활화로의 확장을 고려할 수 있다.
